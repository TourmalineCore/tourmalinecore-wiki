# tourmalinecore-wiki
This repository is the knowledge base of the Tourmaline Core company. The main goal is to store the most important information, guidelines, instructions, useful links, etc. regarding basic technologies and approaches used within the company.

It should help to gather and keep the information in one place in order to increase quality of knowledge transfer between the teams, to decrease the loss of knowledge, and make the new team members onboarding process easier and more efective.  

This knowledge base doesn't include information regarding specific projects. It aims to store the knowledge only about the specific technologies that could be reused in several project. All information regarding projects domain and its sole non-reusable nuances must be stored in projects' repositories.

Anyone from Tourmaline Core team could contribute to this repository. Pull requests must be approved by Aleksandr S. before pushing to the master branch.
